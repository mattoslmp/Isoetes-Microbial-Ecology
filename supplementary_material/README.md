# Supplementary material

This folder is intended to store the supplementary material required to reproduce the manuscript analyses.

## Required tables

- `Table_S1_prinseq_quality_summary.xlsx`: sequencing/quality-control summary and supporting quality metrics.
- `Table_S2_MetaCyc_current_from_latest_paper.xlsx`: current MetaCyc differential abundance table supplied with the latest manuscript package.
- `Table_S3_taxonomic_differential_summary.xlsx`: taxonomic differential abundance results supporting Figure 5 and Figure S1.
- `Table_S4_MetaCyc_enriched_pathways.xlsx`: MetaCyc fold-change workbook used to generate the revised Figure 6.
