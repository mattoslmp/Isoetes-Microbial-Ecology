# Supplementary material manifest

## Tables

| File | Description |
|---|---|
| `Table_S1_prinseq_quality_summary.xlsx` | Sequencing/quality summary table supplied as Table S1. |
| `Table_S2_functional_differential_summary.xlsx` | Functional differential pathway summary supplied as Table S2. |
| `Table_S3_taxonomic_differential_summary.xlsx` | Taxonomic differential abundance summary supplied as Table S3. |
| `Table_S4_MetaCyc_enriched_pathways.xlsx` | MetaCyc enriched pathway fold-change table used to regenerate Figure 6. |

## Figures

| Folder | Contents |
|---|---|
| `figures/main/` | Main manuscript figures, including original and revised Figure 6. |
| `figures/supplementary/diversity/` | Alpha rarefaction and diversity figure outputs. |
| `figures/supplementary/differential_taxa/barplots/` | Differentially abundant taxon barplots. |
| `figures/supplementary/differential_taxa/md_plots/` | NOISeq MD plots for taxonomic contrasts. |
| `figures/supplementary/picrust_pathways/` | PICRUSt pathway boxplot outputs from the original package. |

## Data used by scripts

| Folder | Contents |
|---|---|
| `data/raw/qiime2/` | QIIME2 `.qza` feature, taxonomy and tree artifacts plus metadata. |
| `data/raw/picrust2/` | PICRUSt2 MetaCyc pathway table and Table S4 workbook. |
| `data/processed/` | Tidy CSVs generated from Table S4 for the revised Figure 6. |

For easier browsing in GitHub, spreadsheet sheets were also exported to CSV files in the organized package.
